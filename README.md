## Описание

Добавляет систему фридея в Jailbreak.  
При выдаче фридея заключённый получает усиленные параметры: здоровье, броню, скорость передвижения и визуальный эффект (цвет модели), а также звуковое сопровождение при активации режима.

Механика позволяет централизованно управлять фридеем через Jailbreak Core, обеспечивая единое поведение режима на сервере и упрощая администрирование игрового процесса.
